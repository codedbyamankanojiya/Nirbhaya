# **App Name**: Nirbhaya

## Core Features:

- Emergency SOS: One-tap SOS sends live location, SMS alerts, and initiates auto-call to emergency services and contacts.
- Silent SOS: Secretly activate emergency mode via volume buttons or a hidden interface.
- AI Safety Assistant: An AI tool advises on safety practices based on location and provides real-time emotional support during SOS.
- Guardian Live Tracking: Real-time location tracking for selected guardians with 'Walking With Me' feature and route deviation alerts.
- Threat Detection AI: AI uses sensor data to detect distress events (sudden movements, keywords) and prompts or auto-sends SOS.
- Community Safety Map: AI-generated crime heatmap displays crime hotspots and safe route suggestions with lighting score.
- Fake Call Generator: Generates a fake incoming call to help escape uncomfortable situations.
- Evidence Capture Mode: Silently starts audio and video recording, saves securely, and sends to guardian contacts upon one-tap activation.
- Secure Profile & Authentication: Secure login with biometric authentication, medical info, emergency contacts, and guardian list.

## Style Guidelines:

- Primary color: Lavender (#E6E6FA) for a calming and secure feel.
- Background color: Very light lavender (#F5F5FF).
- Accent color: Soft blue (#B0E2FF) to convey trust and safety.
- Font: 'Inter', a grotesque-style sans-serif with a modern, neutral look for both headlines and body text.
- Use minimalist, clear icons for navigation and feature representation.
- Minimal screens with intuitive navigation to provide clarity and ease of use.
- Subtle animations on interactive elements to enhance user experience.
